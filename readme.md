Documentation
=============

This vagrant box allows you to boot up a Linux-Nginx-PHP-FPM-MySQL server. (LEMP)
Specifically, this includes:
+	Nginx
+ PHP-FPM
+ PHP5/PHP (various modules)
+ MySQL
+ phpMyAdmin
